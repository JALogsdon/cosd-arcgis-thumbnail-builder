About

Building clear, consistent thumbnails helps users quickly understand ArcGIS Online items before they click. This tool streamlines thumbnail creation for the City of San Diego and just makes things a little prettier ;).

City Design Standards: https://www.sandiego.gov/communications/design.

Query Parameters

You can make the tool re-usable by adding the query parameters below to your URL.

See Example

Parameter Description Values Example
titleColor Title background color RGBA (r,g,b,a) 0,255,0,0.8
sidebarColor Sidebar background color RGBA (r,g,b,a) 0,0,255,0.8
category Valid item category label String Web Map
background URL to background image String (URL) https://path.to/background.jpg
logo URL to logo image String (URL) https://path.to/logo.png
